Game Manager¶

The Game Manager.

This module contains the game manager which keeps track of the active scene (i.e. intro, menu, level 1, etc.), switches between scenes, etc.

class game_manager.GameManager(game_opts)[source]¶

The game manager class is responsible for keeping track of the active scene (i.e. intro, menu, level 1, etc), changing between scenes, etc. It’s a borg since we need a shared state for all instances.
